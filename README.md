# Autonomous Sumo Robot Kit

Build, program, and compete with a complete **Autonomous Sumo Robot Kit** designed for robotics enthusiasts, students, makers, and competitions.
This kit provides everything needed to assemble a high-performance autonomous sumo robot featuring opponent detection, edge sensing, high-torque drive motors, and a modular 3D-printed chassis.
It is designed for educational activities, robotics workshops, competitions, and advanced engineering projects, combining powerful hardware with an easy-to-assemble structure suitable for both learning and competitive robot development.

---

## Key Features

✔ Autonomous opponent detection using infrared sensors  
✔ Dohyo edge detection and avoidance system  
✔ High-torque dual motor drive system  
✔ Competition-ready mechanical design  
✔ Modular 3D-printed chassis structure  
✔ Expandable Arduino-based control platform  
✔ Optimized for wooden (non-magnetic) sumo arenas  
✔ Suitable for education, workshops, and robotics competitions  

---

## Kit Includes

### Controller & Electronics
- Arduino UNO Q Controller Board  
- USB Type-C cable  
- Li-ion charger  
- 3S Li-ion battery pack  
- Arduino UNO Terminal Adapter Expansion Board  

### Motor System
- 2 × BTS7960 43A High-Power H-Bridge Motor Drivers (IBT-2)  
- 2 × Titan 12V DC Metal Gear Motors (200 RPM, 60:1 Gear Ratio)  
- 1 Pair × JS5230 Aluminum Silicone Wheels (52 × 30 mm)  

### Sensors
- 3 × E18-D80NK Infrared Obstacle Sensors (10–80 cm range)  
- 2 × QTR-1A Edge Detection Sensors (QRE1113GR based)  

### Controls & Indicators
- 12V Illuminated LED Push Button Switch  
- Traffic Light LED Indicator Module (Red / Yellow / Green)  
- ON/OFF Rocker Switch  

### Wiring & Connectors
- Lever Wire Connector (2 IN / 6 OUT)  
- Lever Wire Connectors (1 IN / 1 OUT)  
- Jumper wire set  
- Battery connection wires  

### Mechanical Parts
- 3D-printed Sumo Robot chassis (4 parts)  
- M2 screws ×10  
- M2 nuts ×10  
- M3 screws ×12  
- 2 screwdrivers  
- 1 Allen key
